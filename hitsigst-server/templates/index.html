Wie is de Hitsigste?

Hitsigst, a project by Mark van der Schoot and Max Crone, is a Christmas present for Mark's parents (who really enjoying playing a very similar game that we shall not name).

Hoi Peter Paul en Steffie!

If you too want to play Hitsigst, trek ons aan de spreekwoordelijke mouw in real life! We might have a few extra sets of cards lying around. And we can always play together!

This game is strictly for personal use. We are not affiliated with the Jumbo Group. Code is all open source.
